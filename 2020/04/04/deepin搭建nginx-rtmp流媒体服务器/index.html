第一步：编译安装Ｎginx

1.安装Ｎginx依赖

  1. 安装gcc g++的依赖库
    sudo apt install build-essential
    sudo apt install libtool

  2. 安装pcre依赖库
    sudo install libpcre3 libpcre3-dev

  3. 安装zlib依赖库
    sudo apt install zlib1g-dev

  4. 安装SSL依赖库
    sudo apt install openssl libssl-dev
